# DokoScan
<img src="https://user-images.githubusercontent.com/31412017/209068085-e6f6d655-76f5-43f8-b956-6e7a31ad72e5.png" alt="output (1)" width="50%" />

Dokoscan is an iOS application that leverages the RoomPlan API to scan rooms and generate 3D floor plan models. Users can enhance their floor plans by adding customizable colored labels and text to furniture, making it easily recognizable and usable for everyone.

## Features

- **Room Scanning**: Utilize the RoomPlan API to scan and create 3D models of rooms.
- **Customizable Labels**: Add colored labels and text to furniture for easy identification.
- **User-Friendly Interface**: Intuitive design for seamless navigation and interaction.

## Requirements

- iOS 15.0 or later
- Xcode 13 or later
- Swift 5.0 or later

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dokoscan.git
   cd dokoscan
Open the project in Xcode.
Build and run the application on a compatible iOS device or simulator.

## Usage
Launch the app and grant camera permissions.
Scan a room using the built-in scanning feature.
Once the scan is complete, customize furniture with labels and text.
Save or share your 3D floor plan model as needed.
